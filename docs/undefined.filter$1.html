function(patterns, opts, `patterns`, `opts`)

Returns a function that takes a glob pattern or array of glob patterns to be used with Array#filter(). (Internally this function generates the matching function using the [matcher] method).

Parameters

  1. patterns {*}

  2. opts {*}

  3. `patterns` {String | Array}

    Can be a glob or array of globs.

  4. `opts` {Options}

    Options to pass to the [matcher] method.

Returns

{function()}

Filter function to be passed to Array#filter().

var fn = mm.filter('[a-c]');
['a', 'b', 'c', 'd', 'e'].filter(fn);
//=> ['a', 'b', 'c']
