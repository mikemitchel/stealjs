function(fp, pattern, opts, `fp`, `pattern`, `opts`)

Returns true if the filepath contains the given pattern. Can also return a function for matching.

Parameters

  1. fp {*}

  2. pattern {*}

  3. opts {*}

  4. `fp` {String}

  5. `pattern` {String}

  6. `opts` {Object}

Returns

{Boolean}

isMatch('foo.md', '*.md', {});
//=> true

isMatch('*.md', {})('foo.md')
//=> true
