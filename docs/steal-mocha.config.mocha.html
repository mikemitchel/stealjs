mocha

  • page
steal-mocha.config.mocha  

The mocha option specifies the user-interface passed into mocha.setup(). This option is passed through Steal config like:

<script src="./node_modules/steal/steal.js" mocha="bdd"></script>

The following are the possible values:

  • bdd
  • tdd
  • qunit
  • exports

See mocha's documentation for more on the user-interfaces.
